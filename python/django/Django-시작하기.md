## Django 시작하기

### 개발 환경: Visual Studio Code
```
python3 -m pip install django
```

### 다양한 명령어 리스트 확인
```
django-admin
```

### 현재 디렉토리에 ‘myproject’ 라는 프로젝트 생성
```
django-admin startproject myproject .
```

### Run (기본포트 http://127.0.0.1:8000/ )
```
Run (기본포트 http://127.0.0.1:8000/ )
```

```
포트 변경(8888)
```
